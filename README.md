# 🚀 AI Resume Analyzer with RAG (Endee Ready)

## 🎯 Overview
This project is an AI-powered Resume Analyzer using Retrieval Augmented Generation (RAG).

## 💡 Features
- Resume understanding using embeddings
- Semantic search using vector similarity
- Question answering on resume
- Job and skill insights

## 🧠 How It Works
1. Resume is split into chunks
2. Each chunk is converted into embeddings
3. Stored in vector database (Endee concept)
4. User queries are matched with relevant chunks
5. System returns best answer

## 🛠️ Tech Stack
- Python
- Sentence Transformers
- NumPy
- Vector Database (Endee-ready)
